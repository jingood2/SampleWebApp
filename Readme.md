# Migration Demo WebApplication for Session Replication

## Introduction
Tomcat 기반의 Session Replication 으로 구성된 웹서비스의 마이그레이션 데모를 위한 Web Application

## Environment
- Framework : Spring 3.2.3
- openJDK 1.8.0_212

## Build
OS 환경에따라 Build는 아래와 같이 수행합니다. 
### Linux
```$xslt
./mvnw clean package
```
### Windows
```$xslt
mvnw.cmd clean install
```
